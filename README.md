# InvestTracker

Это приложение для записи операций по инвестициям (в большей части облигаций) и отслеживания результатов по выплатам купонного дохода. 
Приложение создаётся для личного пользования, но доступ открыт всем, кто также как и я тщательно следит за своими операциями.
Оно предназначено для инвесторов, которые хотят получить статистику своих операций, что им приносут сейчас или принесут потом те бумаги, которые они купили.

## Стек технологий
* **Язык программирования:** C#
* **Платформа:** .NET 4.8
* **GUI:** WinForms
* **База данных:** PostgreSQL

## Функциональность

Список основных возможностей приложения:
* Добавление записей о каждой операции (пополнение счёта, покупка облигации, выплата купонов, погашение облигации) в таблицу, из которой идёт сохранение в БД 
* Выгрузка всех операций и статистики в Excel-файл или .csv-файл
* Просмотр суммы всех выплаченных купонов с выбором даты или диапазона дат. Возможность просмотра купонов по месяцам
* Интеграция с МосБиржей для просмотра подробной информации о бумаге с помощью ISIN кода в таблице с списком текущих облигаций
* Инфографика с бумагами по наибольшей сумме выплаченных купонов, а также другие виды графиков
* Полная информация о погашенных облигациях (сколько всего было выплачено купонов, какая итоговая прибыль от бумаги с учётом всех трат)
* Подробный отчёт операций по выбранным датам и не только
